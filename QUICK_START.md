## 🚀 启动方式

### 方法一：使用启动脚本（推荐）

**Windows:**
```bash
start.bat
```

**Linux/Mac:**
```bash
bash start.sh
```

### 方法二：手动启动

```bash
# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm start
```

服务器将在 http://localhost:3000 启动

## 🎨 样式特点

### 色彩
- 主色：#007AFF（Apple 蓝）
- 成功：#34C759（Apple 绿）
- 警告：#FF9500（Apple 橙）
- 背景：#F5F5F7（浅灰）
- 文字：#1D1D1F（深灰）

### 圆角
- 小：8px
- 中：12px
- 大：16px
- 超大：20px

### 阴影
- 轻柔：0 2px 8px rgba(0,0,0,0.04)
- 中等：0 8px 24px rgba(0,0,0,0.06)
- 深重：0 20px 60px rgba(0,0,0,0.3)
